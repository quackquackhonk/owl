# Owl Language


Owl is a functional language compiled in Haskell.

## Usage

With `stack` installed, you can run the owl complier, "pellets" with the following command:

    stack run -- <filename>
    
If you wish to launch an Owl REPL, use this command instead:

    stack run -- --repl

## Why

I thought it would be fun to write a language in Haskell
